## 2026-06-25（Day 2 · 大迭代 + 双语 + 报告）

**概览**：在已上线版本上做了一大轮打磨与新功能——绿色重塑、支付高潮链、音效、计分爆炸、心理侧写收据、订单/我的、中英双语、月度冲动报告，并推送到 repo。

### 做了什么
- **视觉 / 品牌**
  - 主题色 美团黄 → **品牌绿**；所有背景拍平成**纯色**（去渐变）
  - 引入字体：Courier Prime（打字机体）+ Archivo（厚重体）
- **逛店 / 点餐**
  - 搜索框真能用：「捕捉馋」——输入想吃的 → 直接进对应店并高亮那道菜
  - 分类九宫格改成真菜系，点了能筛选店铺
  - 商家卡推荐语换成搞笑网络热梗；搜索框滚动热梗
  - **每店满减按价位定**（贵店门槛高、减得多）
- **下单高潮（小丑牌式）**
  - 支付方式：Apple Pay 扫脸 / Visa / Mastercard / 微信 + 真音效
  - 美区**小费** + **骑手配送追踪**（真实预计时间 + 搞笑文案）/ **自取走一圈**（自取点=大脑/身体部位）
  - **计分爆炸**：连击 = 折扣来源个数（+店铺专属红包 / 店庆红包，最高 ×8）、震屏、彩虹撒花
  - **Web Audio 音效**（数字滚动=前额叶式干净爬升；其余=多巴胺爆炸）
  - **随机红包奖励**；toast 正/负反馈分色（绿/红）
- **收据 / 分享**
  - 收据=毛边热敏小票（撕齿边 + 条形码 + SPENT 字标）
  - **心理侧写**：按本单选择给挂心理学的毒舌解读（不用红包→「我不配」型损失厌恶…）
  - **一键保存图片**发小红书 / X
- **留存层**
  - 订单页（拦截史，可重看任意收据）/ 我的页（段位 + 战绩 + 战利品墙）
  - **月度冲动报告**（"wrapped"风格可截图长卡：最馋的店 / 最危险时段 / 最大一笔 / 段位）
- **中英双语**：中 | EN 一键切换（可逆翻译，收据 / 报告全英文），菜名店名保留原生
- **真图试点**：菜品/店铺支持 `img` 字段显示真实照片，失败回退 emoji（已挂 5 家旗舰店占位真图）

### 决策 / 备注
- 连击的正路是**加更多折扣来源**（新客立减 / 平台津贴 / 神券…）→ 解锁 ×9 ×10，不是把数字灌大（见 BRAINSTORM §10）
- 折扣靠**小游戏/抽奖赚** + 连击 → 也许之后做成 Steam 小游戏（暂不搞大）
- 真图：先给招牌菜上几张真实感图看转化，再决定要不要全量替换（版权见 BRAINSTORM §9）

## 2026-06-24（Day 1 · 从 0 到上线）

**概览**：从小红书"多巴胺网站"灵感出发，定名 **SPENT**，写设计文档，做出可玩原型，逐步堆到 14 家店 + 选规格 + 美团级仿真，并**首次上线 repo**。

### 做了什么
- **立项**：定位"假装下单机——花掉冲动不花钱"；定名 **SPENT**（spend 过去式 + "I'm spent" 双关）；写 `BRAINSTORM.md`
- **范围**：聚焦食物（外卖流），先做一条赛道做透
- **核心爽链原型**：逛店 → 加购 → 满减/凑单/红包膨胀 → 下单动画 → 转折收据 → 战利品墙
- **多店扩张**：5 → 14 家店（含墨西哥肉类中英名、日料、ShackBurger、W记、CFK、韩料、YoTea、铁板烧、居酒屋自助）
- **选规格系统**：通用的 options 引擎（墨西哥按肉 / 茶饮糖度加料 / 韩式炸鸡一半一半 / 自助套餐多选）
- **美团级仿真 UI**：状态栏、首页/分类、商家卡、店铺页（外送自取/点菜评价/左侧分类）、底部导航
- **转折卡重设计**：从 AI 味卡片 → **「反向收据」热敏小票**（撕齿边 + 印章 + 条形码 + SPENT 字标）
- **「距离你的胃」梗**：自取/配送写成身体部位
- **上线**：推送到 github.com/boxsbraindump/spent

### 决策 / 备注
- 基调红线：**爽是主线，治愈/锐评是彩蛋**，不说教
- 美团仿真是产品的笑点，不能为了"去 AI 感"破坏它——独特设计只放在 SPENT 自己的品牌界面（收据等）
- 商品图先用风格化 emoji（避版权）
